## Установка

1) Скачать архив.  
Это можно сделать здесь по кнопке `Code → Download ZIP`.  
![image](https://github.com/user-attachments/assets/4d5f9ad0-47c8-4a73-8140-019406c58053)

2) Открыть папку с игрой.

3) Из архива нужно переместить только папку `game`.  
Если всё верно, то должна быть папка: `/game/tl/russian`

4) После этого в настройках появится выбор языка.
![image](https://github.com/user-attachments/assets/5b8b1688-940e-4ac4-9f1a-8e11bf797c2e)
